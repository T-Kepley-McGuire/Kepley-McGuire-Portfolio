Projects

Mandelbrot Set Renderer

I created a rendering engine for the mandelbrot set that is lightweight and has a few basic features.

Some of the features in the engine include optimizations in areas of multithreading through a chunking system, block detection to cut down on redundant calculation, algorithmic adjustments that alter the basic equation to better suit computer architecture, and histogram colorgrading.

I invite you to explore the mandelbrot set for yourself using my engine! It's a beautiful and otherworldly experience.
